### 主要性能指标

- 性能指标
  - 有效性
  - 可靠性
- 有效性指标
  - 1. 码元传输速率 RB (传码率, 波特率)
       - 定义 : 没秒传送的码元个数
       - 单位 : 波特 (Baud)
       - 例如 : 1 s 内传输1000 个码元, 则 Rb = 1000 Baud .
       - Rb = 1 / Ts. 1 个码元的时间 是 Ts 秒.
    2. 信息传输速率 Rb (传信率, 比特率)
       - 定义 : 每 秒 传输的比特数( 信息量 )
       - 单位 : bit / s 简记为 b / s 或 bps.
    3. Rb 和 RB 的关系 ------- H 为纽带
       - Rb = RB * H
       - Rb = RB * log 2 M
         - M = 2 : Rb = RB
    4. 频带利用率 
       - 将 B 将 传输 速率 联系起来
         - n = RB / B (Baud / Hz)
         - nb = Rb / B (b / s * Hz -1)
