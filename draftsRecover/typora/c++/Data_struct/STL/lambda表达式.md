# lambda表达式
* 1. 一个lambda表达式表示一个可以调用的代码单元，简单来说，我们可以将其理解为一个未命名的内联函数，
* 2. 形式  [ 捕获列表 ] (参数列表) -> return type { function body }
	- 1.捕获列表用于在该函数中定义局部变量
	- 2.参数列表用于传递参数
		- 	1.1.[var]表示值传递方式捕捉变量var；
          	2.[=]表示值传递方式捕捉所有父作用域的变量（包括this）；
 			3.[&var]表示引用传递捕捉变量var；
			4.[&]表示引用传递方式捕捉所有父作用域的变量（包括this）；
			5.[this]表示值传递方式捕捉当前的this指针。

* 3. lambda函数不支持默认参数
			1.lamuda函数的返回值是一个函数指针
