#### HTTP 报文内的 HTTP 信息

- HTTP 报文
  - 用于 HTTP 协议交互的信息被称为 HTTP 报文。
    - 请求端 ( 客户端 ) 的 HTTP  报文叫做请求报文
    - 响应端 ( 服务器端 ) 的叫做 响应报文
  - HTTP  报文大致可以分为 报文首部和报文主体两块。
